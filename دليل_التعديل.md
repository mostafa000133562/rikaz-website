# دليل تعديل المحتوى بسرعة

## تعديل بيانات الشركة
افتح:
`assets/js/data.js`

ابحث عن:
`company`

يمكن تعديل:
- اسم الشركة
- رقم الواتساب
- العنوان
- رابط الخريطة
- نبذة الشركة
- الرؤية
- الرسالة

## تعديل بيانات المشروع
في نفس الملف ابحث عن:
`projects`

يمكن تعديل:
- اسم المشروع
- الموقع
- المطور
- حالة المشروع
- وصف المشروع
- المميزات

## تعديل الوحدات
في نفس الملف ابحث عن:
`units`

كل وحدة لها بيانات مثل:
- `id`: رقم الوحدة مثل A-01
- `floorName`: الدور
- `netArea`: صافي المساحة
- `roofArea`: مساحة خارجية / سطح
- `status`: حالة الوحدة
- `planImage`: مسار صورة مخطط الوحدة

## تعديل صورة وحدة
استبدل الصورة الموجودة داخل:
`assets/unit-plans/`
بنفس اسم ملف الوحدة، مثل:
`A-01.jpg`
