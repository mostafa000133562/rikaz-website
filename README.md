# موقع ركاز للخدمات العقارية - Static Website

الموقع عربي فقط وبدون Admin Panel أو قاعدة بيانات. يعمل على أي Hosting عادي أو سيرفر ملفات ثابتة.

## طريقة التشغيل المحلي
افتح الملف `index.html` مباشرة، أو شغّل سيرفر بسيط من داخل المجلد:

```bash
python -m http.server 8000
```
ثم افتح:

```text
http://localhost:8000
```

## الملفات المهمة للتعديل لاحقًا

- `assets/js/data.js`  
  يحتوي على بيانات الشركة، المشروع، الوحدات، رقم الواتساب، ورابط Google Maps.

- `assets/css/main.css`  
  ملف التصميم والألوان والخطوط.

- `assets/images/`  
  اللوجو وصورة المشروع الرئيسية.

- `assets/unit-plans/`  
  صور المخططات المقصوصة لكل وحدة.

## ملاحظات

- الأسعار غير ظاهرة في الموقع.
- كل الوحدات حالتها الحالية: متاحة.
- إرسال نموذج التواصل يتم عبر واتساب فقط.
- صفحة الوحدة تعرض مخطط الوحدة المختارة فقط.
- لا توجد روابط سوشيال ميديا حاليًا، ويمكن إضافتها لاحقًا.
